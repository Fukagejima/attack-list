## 6. スコアの計算方法

### 業界スコア（0〜100点）
```
Customer（市場性）  × 40%
Competitor（競合少なさ） × 30%
Company（自社適合）  × 30%
= 業界総合スコア
```

### 企業スコア（0〜100点）
```
ICP適合度（業種・規模の合致）    × 40%
ソリューションフィット（課題合致） × 35%
競合飽和度（参入余地）           × 25%
= 企業総合スコア
```

## 9. Claude Code・Skills・MCP との関係

このアプリは **Claude Code（ターミナルで動くAIアシスタント）** を使って開発・改修されました。

### Claude Code とは
ターミナル上で動く AI コーディングアシスタント。
コードの読み書き・実行・ファイル操作・ウェブ検索などを自律的に行える。

### Skills（スキル）
Claude Code に追加できる専門機能。
このプロジェクトでは以下のスキルが活用されました：

| スキル名 | 用途 |
|---------|------|
| `claude-api` | Anthropic SDK の使い方・プロンプトキャッシュ設計 |
| `anthropic-skills:xlsx` | Excel出力ロジック（openpyxl）の設計 |
| `simplify` | 書いたコードの品質レビュー・リファクタリング |

### MCP（Model Context Protocol）
Claude Code が外部ツールを操作するための仕組み。
このプロジェクトの開発中に使用されたMCP：

| MCP名 | 用途 |
|-------|------|
| `computer-use` | 画面のスクリーンショット確認、ファイル操作 |
| `Claude in Chrome` | ブラウザ上のStreamlit画面確認 |
| `filesystem` | プロジェクトフォルダのファイル読み書き |

**注意：** アプリ本体（Streamlit）は MCP を使っていません。
Claude API を Python SDK 経由で直接呼び出しています。

## 10. プロンプトキャッシュとは（コスト削減の仕組み）

Claude API の呼び出しにはトークン（文字数）に応じた料金がかかります。
このアプリでは **プロンプトキャッシュ** を使ってコストを削減しています。

```python
# システムプロンプトをキャッシュ対象に指定
system=[{
    "type": "text",
    "text": SYSTEM_PROMPT,
    "cache_control": {"type": "ephemeral"},  ← これがキャッシュ指定
}]
```

同じシステムプロンプトを繰り返し送るとき（バッチ分析の2回目以降）、
キャッシュから読み込むので入力コストが **約90%削減** されます。

## 11. よくある質問

**Q. どのくらいAPIコストがかかる？**
1回の全業界分析（30業界）＋アタックリスト生成（3業界×10社）で
概算 **$0.10〜0.30（約15〜45円）** 程度。

**Q. アタックリストの企業情報は正確？**
Claude が公開情報をもとに生成した **推定情報** です。
IRデータや公式サイトで確認した情報には `data_confidence: high`、
推定が多い場合は `medium` や `low` が付きます。

**Q. ウェブ補強をONにするとどう変わる？**
DuckDuckGo等からリアルタイムの市場情報を取得してプロンプトに付加します。
精度が若干向上しますが、処理時間が1〜2分延びます。

**Q. 生成したExcelはどこに保存される？**
プロジェクトフォルダ内の `output/` フォルダに
`アタックリスト_20260509_143022.xlsx` のような形式で保存されます。
